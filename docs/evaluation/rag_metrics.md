# RAG 效果评估指标

本文件用于定义 学科智引：基于RAG的垂类大模型助学助教平台 的 RAG 问答效果评估指标。当前只提供指标口径和待填写表格，不包含伪造评分、响应时间或用户反馈。

## 1. 答案正确率

将系统输出与人工标准答案、课程教师判断或授权课程资料进行对比，检查公式、概念、推导、单位和适用条件是否正确。

## 2. 引用命中率

检查回答是否引用了与问题相关的本地课程资料或网络补充资料。引用需要与问题相关，不能只返回无关片段。

## 3. 结构完整率

对 `step_solution` 任务，检查是否包含：

- 题目信息提取；
- 物理模型判断；
- 分步推导；
- 结果；
- 易错点；
- 引用来源。

## 4. 可视化触发率

对斜抛运动类问题，检查系统是否返回 `visualization_hint`，并且前端是否能据此生成轨迹图或轻量动画。

## 5. 本地知识库优先率

统计回答中优先使用 local citation 的比例。正式参赛展示应尽量以本地课程资料为主要依据，网络资料只作为补充参考。

## 6. 低依据提示率

当本地和网络检索均无可靠结果时，检查系统是否明确提示“当前知识库和网络检索中未找到可靠依据”，而不是编造来源或假装引用课程资料。

## 7. 平均响应时间

团队后续真实运行后填写。不要在未测试前填写模拟响应时间。
