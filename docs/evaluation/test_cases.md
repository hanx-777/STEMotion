# 参赛效果验证测试案例

本目录用于 XH202620 参赛效果验证，记录典型输入、预期输出结构、人工检查点和后续真实用户反馈材料。

注意：本目录中的用户反馈和效果报告当前为模板或测试设计，不包含伪造的真实用户反馈。

## Case 01：斜抛运动分步解题

- 任务类型：`step_solution`
- 测试目标：验证系统能基于大学物理力学知识库完成斜抛运动分步推导、结果计算、易错点提示、引用来源和可视化参数生成。
- 输入问题：一个小球以 20 m/s 初速度、30 度角斜向上抛出，忽略空气阻力，求最大高度和水平射程。
- 预期输出结构：题目信息提取、物理模型判断、分步推导、计算结果、易错点、引用来源、可视化参数。
- 验证方式：与人工标准答案对比，检查公式、单位、适用条件、引用来源和 `projectile_motion` visualization hint。

详见 [physics_case_01_projectile_motion.md](physics_case_01_projectile_motion.md)。

## Case 02：圆周运动概念讲解

- 任务类型：`knowledge_qa`
- 测试目标：验证系统能解释“速度大小不变但有加速度”的物理概念，并指出速度方向变化和向心加速度。
- 输入问题：为什么匀速圆周运动速度大小不变但仍然有加速度？
- 预期输出结构：概念解释、速度方向变化、向心加速度、公式、易错点、引用来源。
- 验证方式：检查概念准确性、公式 `a = v^2 / r`、引用来源和误区提醒。

详见 [physics_case_02_circular_motion.md](physics_case_02_circular_motion.md)。

## Case 03：学生错误答案诊断

- 任务类型：`misconception_diagnosis`
- 测试目标：验证系统能识别学生答案中的错误公式，解释错因，并给出复习建议。
- 输入问题：学生答案：斜抛最大高度 H = v0^2 / 2g。请判断是否正确并说明原因。
- 预期输出结构：错误定位、正确公式、错因分析、复习建议、引用来源。
- 验证方式：检查是否指出应使用竖直分速度 `v0 sin θ`，而不是总初速度 `v0`。

详见 [physics_case_03_misconception.md](physics_case_03_misconception.md)。

## 可选 Case 04：教师课堂演示生成

- 任务类型：`teacher_prep`
- 测试目标：验证系统能生成面向教师的课堂讲解方案、演示步骤和可视化参数建议。
- 输入问题：请设计一个讲解斜抛运动最大高度和射程的 5 分钟课堂演示。
- 预期输出结构：教学目标、导入问题、讲解步骤、可视化演示建议、课堂提问、引用来源。
- 验证方式：由教师检查是否适合课堂演示，后续可纳入真实用户反馈。
