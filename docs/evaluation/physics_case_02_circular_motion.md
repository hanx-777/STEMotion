# Case 02：圆周运动概念讲解

## 案例名称

圆周运动概念讲解

## 任务类型

`knowledge_qa`

## 测试问题

为什么匀速圆周运动速度大小不变但仍然有加速度？

## 预期输出

- 概念解释；
- 速度方向变化；
- 向心加速度；
- 公式 `a = v^2 / r`；
- 易错点；
- 引用来源。

## 人工标准要点

- 速度是矢量，包含大小和方向。
- 匀速圆周运动的速率不变，但速度方向不断变化。
- 加速度描述速度矢量变化率，因此速度方向变化也意味着存在加速度。
- 该加速度指向圆心，称为向心加速度。
- 向心加速度大小为：

```text
a = v^2 / r
```

也可写为：

```text
a = ω^2 r
```

## 准确性检查点

- 是否明确区分“速率不变”和“速度矢量变化”；
- 是否说明加速度方向指向圆心；
- 是否给出 `a = v^2 / r`；
- 是否提示常见误区：速度大小不变不等于没有加速度；
- 是否给出引用来源。
