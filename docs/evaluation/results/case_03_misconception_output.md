## 标准答案

该答案通常不完整或错误。`H = v0² / 2g` 只适用于竖直上抛，或 `v0` 已明确表示竖直分速度的情况。斜抛运动应先分解初速度：

```text
v0y = v0 sinθ
H = (v0 sinθ)² / 2g
```
