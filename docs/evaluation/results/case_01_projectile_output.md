## 标准答案

人工标准答案：

```text
v0y = v0 sinθ
H = v0y² / 2g
T = 2v0y / g
R = v0² sin(2θ) / g
```

代入 `v0 = 20 m/s`，`θ = 30°`，`g = 9.8 m/s²`：

- `v0y = 20 × sin30° = 10 m/s`
- `H = 10² / (2 × 9.8) ≈ 5.10 m`
- `R = 20² × sin60° / 9.8 ≈ 35.35 m`
