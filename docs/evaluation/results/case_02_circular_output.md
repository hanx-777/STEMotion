## 标准答案

匀速圆周运动中，速度大小不变，但速度是矢量，方向沿轨迹切线不断变化。加速度描述速度矢量随时间的变化率，因此即使速率不变，也存在指向圆心的向心加速度。向心加速度大小为：

```text
a = v² / r
```

方向指向圆心。
