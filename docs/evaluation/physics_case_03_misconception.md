# Case 03：学生错误答案诊断

## 案例名称

学生错误答案诊断

## 任务类型

`misconception_diagnosis`

## 测试问题

学生答案：斜抛最大高度 H = v0^2 / 2g。请判断是否正确并说明原因。

## 预期输出

- 错误定位；
- 正确公式；
- 错因分析；
- 复习建议；
- 引用来源。

## 人工标准要点

学生答案不完整或不正确。斜抛最大高度只由竖直方向分速度决定，不能直接把总初速度 `v0` 代入竖直上抛公式。

正确处理方式：

```text
v0y = v0 sin θ
H = v0y^2 / 2g
H = v0^2 sin^2 θ / 2g
```

如果题目没有角度或竖直分速度，则无法唯一计算最大高度。

## 错因分析

- 混淆总初速度和竖直分速度；
- 忽略速度分解；
- 未说明公式适用条件；
- 把竖直上抛公式直接套到斜抛问题。

## 复习建议

- 复习矢量分解；
- 复习水平和竖直方向独立运动；
- 练习从题目中提取 `v0`、`θ`、`g` 和落地高度条件；
- 每次使用公式前说明适用条件。

## 准确性检查点

- 是否判断学生答案不正确或条件不足；
- 是否给出 `H = v0^2 sin^2 θ / 2g`；
- 是否解释 `v0y` 的作用；
- 是否给出复习建议；
- 是否给出引用来源。
