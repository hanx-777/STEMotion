# STEMotion 功能分级

## 一级：参赛主应用层

### 学生助学中心 (`/rag`)

面向大学生的物理力学智能学习助手，基于课程知识库提供可追溯的结构化回答。

- **知识讲解**：解释物理概念、公式推导和物理直觉
- **分步解题**：提取题目信息、判断物理模型、分步推导计算结果
- **错因诊断**：分析错误答案，指出公式误用和复习建议

### 教师助教中心 (`/rag`)

面向教师的备课和课堂演示辅助工具。

- **课堂备课**：生成教学目标、课堂导入、互动提问和课后练习
- **演示设计**：生成课堂可视化演示流程和参数建议
- **练习生成**：围绕当前知识点生成练习题和易错点提醒

### 可视化演示中心 (`/rag`)

展示 STEMotion 运动可视化特色能力。

- **斜抛运动轨迹**：根据初速度、角度和重力加速度生成轨迹图
- **动态演示**：页面内轻量动画面板
- **参数卡片**：展示最大高度、水平射程和飞行时间

## 二级：可信知识底座层

### 学科知识库

每个学科 Skill 配置独立的知识库，支持课程讲义、教材章节等本地文档的分块索引和混合检索（TF-IDF 词法 + 可选 Embedding 语义）。

### 可追溯 RAG 引擎

所有回答均附带 `[Lx]` 本地课程资料和 `[Wx]` 网络补充资料的可追溯引用标记，支持引用定位和原文片段展开。

- 本地课程资料为优先可信来源
- 网络补充资料仅在本地证据不足时触发，作为补充参考

## 三级：学科扩展层

### 学科 Skill 管理

支持多学科 Skill 配置（physics_mechanics、advanced_math、chemistry、computer_science），每个 Skill 包含：

- 系统提示词和回答模板
- 知识库文件和检索配置
- 工具能力标签和回答规范

## 四级：系统支撑层

### 深度交互实验生成 (`/deep-interaction`)

教师-AI 共创交互实验，支持多 Agent 反馈循环生成交互式 HTML 小组件。

### 参赛验证与材料管理

- 交互库 (`/interactions`)：已保存的 artifact 管理
- 模型与 API 设置 (`/settings`)：模型配置和 API Key 管理
- 实验工作台 (`/`)：快速实验展示
