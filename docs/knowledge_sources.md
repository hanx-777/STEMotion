## 使用原则

- 本地知识库是系统优先可信来源。
- 网络检索只作为补充参考来源。
- 系统回答不得把网络检索结果伪装成本地课程资料。
- 如果知识库和网络检索中没有可靠依据，应明确提示“当前知识库和网络检索中未找到可靠依据”。
- 正式参赛展示时，应明确区分“示例知识库”和“真实课程资料”。
- 引入教材、论文、讲义或题库时，应确认授权、引用规范和可公开范围。

## 网络检索边界

当前系统支持可选网络检索增强。若未配置真实搜索 API Key，系统会优雅降级，不影响本地 RAG 流程。测试或演示中使用的 MockWebSearchProvider 只能说明流程可用，不应被描述为真实联网检索结果。
