# 3 分钟演示脚本

## 0:00 - 0:20 项目定位

打开 `/rag`，说明 STEMotion Physics Skill 是面向高校大学物理力学课程的可追溯 RAG 智能助学与助教系统。

强调当前参赛版本聚焦大学物理力学，不是泛化问答页面。

## 0:20 - 0:50 Skill 与知识库

展示默认学科 `physics_mechanics`。

点击“查看 Skill 配置”，展示：

- 学科范围；
- 知识库状态；
- 能力标签；
- 回答规范；
- 是否启用网络检索。

说明本地课程知识库是优先可信来源，网络检索仅作为补充参考。

## 0:50 - 1:40 斜抛运动分步解题

点击典型案例“斜抛运动分步解题”，发起请求。

展示结构化回答：

- 题目信息提取；
- 物理模型判断；
- 分步推导；
- 结果；
- 易错点；
- 引用来源。

强调回答会标注来源，不能把网络补充资料伪装成本地课程资料。

## 1:40 - 2:10 可视化

展示 `visualization_hint` 和 SVG 轨迹图。

说明 STEMotion 不只是问答，还能把物理过程可视化，服务课堂讲解和学生理解。

## 2:10 - 2:35 教师备课

切换任务类型为“教师备课”。

展示系统可以生成课堂讲解方案、演示参数和教师提示，支持助教场景。

## 2:35 - 3:00 可扩展性

展示学科切换：高等数学、大学化学、程序设计与数据结构。

说明当前参赛版本做深大学物理力学，但架构可扩展到其他 STEM 学科。原有深度交互生成、多 Agent 评审和本地交互库作为系统扩展能力保留。
