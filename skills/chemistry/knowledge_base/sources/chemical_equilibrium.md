# 化学平衡基础

可逆反应在一定条件下，正反应速率和逆反应速率相等时，体系达到化学平衡。此时各组分浓度保持不变，但微观上正逆反应仍在进行，因此化学平衡是动态平衡。

对于一般反应 \(aA+bB \rightleftharpoons cC+dD\)，平衡常数可表示为
\[
K=\frac{[C]^c[D]^d}{[A]^a[B]^b}
\]
平衡常数只与温度有关，浓度变化会改变平衡位置，但在温度不变时不会改变平衡常数。

勒夏特列原理指出，当外界条件改变时，平衡会向减弱这种改变的方向移动。例如升高反应物浓度通常使平衡向生成物方向移动；升高温度会使平衡向吸热方向移动。

常见误区包括：把平衡状态理解为反应停止；认为加入催化剂会改变平衡常数；没有区分平衡移动和平衡常数变化。实验中处理酸碱或挥发性试剂时应佩戴护目镜并保持通风。
