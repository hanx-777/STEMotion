# 大学化学典型题库

## 化学平衡题组

**题1：** 在密闭容器中，\(N_2(g) + 3H_2(g) \rightleftharpoons 2NH_3(g)\) 达到平衡。增大压强，平衡如何移动？

**题2：** 某反应的 \(K = 1.0\times10^{-3}\)，初始浓度 \([A] = 0.10\,\text{mol/L}\)，\([B] = 0\)。求平衡时各物质浓度。

## 酸碱平衡题组

**题3：** 计算 0.010 mol/L HCl 溶液的 pH。

**题4：** 计算 0.10 mol/L CH\(_3\)COOH 溶液的 pH（\(K_a = 1.8\times10^{-5}\)）。验证近似条件。

**题5：** 将 0.10 mol/L NaOH 溶液与 0.10 mol/L CH\(_3\)COOH 溶液等体积混合，求混合液 pH。

## 缓冲与滴定题组

**题6：** 配制 pH = 5.0 的缓冲溶液，选择什么弱酸/共轭碱体系？各物质浓度比为多少？

**题7：** 用 0.1000 mol/L NaOH 滴定 25.00 mL 0.1000 mol/L HCl，求加入 12.50 mL 和 25.00 mL NaOH 时的 pH。

## 热化学题组

**题8：** 已知 \(C(s) + O_2(g) \to CO_2(g)\) 的 \(\Delta H = -393.5\,\text{kJ/mol}\)，\(CO(g) + \frac{1}{2}O_2(g) \to CO_2(g)\) 的 \(\Delta H = -283.0\,\text{kJ/mol}\)。求 \(C(s) + \frac{1}{2}O_2(g) \to CO(g)\) 的 \(\Delta H\)。

## 参考来源与整理说明

本文为大学化学课程的原创整理题库，覆盖化学平衡、酸碱平衡、缓冲溶液和热化学的典型计算题。
