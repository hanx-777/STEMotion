# 化学平衡

## 动态平衡

化学平衡是动态平衡：正反应和逆反应仍在进行，但速率相等，宏观上各物质浓度不再变化。

## 平衡常数

对于反应 \(aA + bB \rightleftharpoons cC + dD\)，平衡常数：
\[
K = \frac{[C]^c [D]^d}{[A]^a [B]^b}
\]

\(K\) 只与温度有关，与浓度、压强无关。\(K\) 越大，平衡越偏向产物。

## 反应商与平衡移动

反应商 \(Q\) 的表达式与 \(K\) 相同，但使用任意时刻的浓度。
- \(Q < K\)：反应正向进行。
- \(Q = K\)：达到平衡。
- \(Q > K\)：反应逆向进行。

## 勒夏特列原理

改变平衡系统的条件，平衡向减弱该改变的方向移动：
- 增加反应物浓度 → 平衡正移。
- 升高温度 → 平衡向吸热方向移动。
- 增大压强 → 平衡向气体分子数减少的方向移动。

注意：催化剂不影响平衡位置，只加快达到平衡的速度。

## 常见误区

- 认为平衡时正反应停止——平衡是动态的。
- 将 \(K\) 与 \(Q\) 混淆。
- 忘记温度是唯一能改变 \(K\) 的因素。
