# 高等数学典型题库

## 极限题组

**题1：** 求 \(\lim_{x\to 0} \frac{e^x - e^{-x} - 2x}{x - \sin x}\)。

**题2：** 求 \(\lim_{x\to 0^+} x^x\)。

## 导数题组

**题3：** 分析 \(f(x) = x e^{-x^2}\) 的单调区间、极值点、凹凸区间和拐点。

**题4：** 求曲线 \(y = \ln x\) 在 \(x = 1\) 处的切线方程。

## 积分题组

**题5：** 计算 \(\int x e^x \, dx\)。

**题6：** 计算 \(\int_0^1 \sqrt{1-x^2} \, dx\)。

## 级数题组

**题7：** 判断 \(\sum_{n=1}^{\infty} \frac{n}{2^n}\) 的收敛性并求和。

**题8：** 将 \(f(x) = \frac{1}{1-x}\) 展开为幂级数。

## 微分方程题组

**题9：** 求方程 \(y' + y = e^x\) 的通解。

**题10：** 求方程 \(y'' - 3y' + 2y = 0\) 的通解。

## 参考来源与整理说明

本文为高等数学课程的原创整理题库，覆盖极限、导数、积分、级数和微分方程的典型计算题。
