# 导数与应用

## 导数定义

函数 \(f(x)\) 在 \(x = a\) 处的导数定义为：
\[
f'(a) = \lim_{\Delta x \to 0} \frac{f(a + \Delta x) - f(a)}{\Delta x}
\]

几何意义：曲线 \(y = f(x)\) 在点 \((a, f(a))\) 处切线的斜率。

## 求导法则

和差法则：\((f \pm g)' = f' \pm g'\)
积法则：\((fg)' = f'g + fg'\)
商法则：\((f/g)' = (f'g - fg')/g^2\)
链式法则：\([f(g(x))]' = f'(g(x)) \cdot g'(x)\)

## 常用导数

\[
(x^n)' = nx^{n-1}, \quad (\sin x)' = \cos x, \quad (\cos x)' = -\sin x
\]

\[
(e^x)' = e^x, \quad (\ln x)' = \frac{1}{x}, \quad (a^x)' = a^x \ln a
\]

## 中值定理

罗尔定理：若 \(f(a) = f(b)\)，则存在 \(\xi \in (a,b)\) 使 \(f'(\xi) = 0\)。

拉格朗日中值定理：存在 \(\xi \in (a,b)\) 使 \(f'(\xi) = \frac{f(b)-f(a)}{b-a}\)。

## 洛必达法则

对于 \(\frac{0}{0}\) 或 \(\frac{\infty}{\infty}\) 型不定式：
\[
\lim \frac{f(x)}{g(x)} = \lim \frac{f'(x)}{g'(x)}
\]

前提：右侧极限存在（或为无穷）。

## 常见误区

- 在使用链式法则时漏掉内层导数。
- 在使用洛必达法则前不检查是否为不定式。
- 混淆导数存在与函数连续的关系（可导必连续，连续不一定可导）。
