# 级数基础

## 数项级数

级数 \(\sum_{n=1}^{\infty} a_n\) 收敛的定义：部分和序列 \(S_N = \sum_{n=1}^{N} a_n\) 有极限。

必要条件：\(\lim_{n\to\infty} a_n = 0\)（不满足则发散，但满足不一定收敛）。

## 收敛判别法

比较判别法：若 \(0 \leq a_n \leq b_n\)，\(\sum b_n\) 收敛则 \(\sum a_n\) 收敛。
比值判别法：\(\lim_{n\to\infty} |a_{n+1}/a_n| = \rho < 1\) 则收敛。
根值判别法：\(\lim_{n\to\infty} \sqrt[n]{|a_n|} = \rho < 1\) 则收敛。
积分判别法：将级数与对应的积分比较。

## 交错级数

莱布尼茨判别法：若 \(a_n\) 单调递减趋于零，则 \(\sum (-1)^n a_n\) 收敛。

## 幂级数

\[
\sum_{n=0}^{\infty} c_n (x - a)^n
\]

收敛半径 \(R\)：当 \(|x - a| < R\) 时绝对收敛，\(|x - a| > R\) 时发散。
\[
R = \lim_{n\to\infty} \left|\frac{c_n}{c_{n+1}}\right|
\]

## 泰勒展开

\[
f(x) = \sum_{n=0}^{\infty} \frac{f^{(n)}(a)}{n!}(x-a)^n
\]

常用展开：\(e^x = \sum x^n/n!\)，\(\sin x = \sum (-1)^n x^{2n+1}/(2n+1)!\)。

## 常见误区

- 将必要条件当作充分条件：\(a_n \to 0\) 不保证收敛。
- 在求收敛半径时使用错误的公式。
- 混淆绝对收敛和条件收敛。
