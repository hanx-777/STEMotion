# 导数基础

函数 \(f(x)\) 在点 \(x_0\) 处的导数定义为
\[
f'(x_0)=\lim_{\Delta x\to 0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}
\]
如果该极限存在，则称函数在 \(x_0\) 处可导。

导数的几何意义是曲线 \(y=f(x)\) 在点 \(x_0\) 处切线的斜率。导数也可表示瞬时变化率。

常见求导法则包括：常数倍法则、和差法则、乘积法则、商法则和复合函数链式法则。使用链式法则时，需要先识别外层函数和内层函数。

常见错误包括：把连续误认为一定可导；忘记使用链式法则；对商法则分子符号处理错误；忽略函数定义域。
