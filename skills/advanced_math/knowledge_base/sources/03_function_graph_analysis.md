# 函数图像分析

## 分析流程

利用导数研究函数图像的完整流程：

1. 确定定义域和值域。
2. 判断奇偶性（对称性）。
3. 求一阶导数 \(f'(x)\)，确定单调区间和极值点。
4. 求二阶导数 \(f''(x)\)，确定凹凸区间和拐点。
5. 求渐近线（水平、竖直、斜渐近线）。
6. 计算关键点的函数值，列表描点画图。

## 单调性判据

设 \(f(x)\) 在区间 \(I\) 上可导：
- \(f'(x) > 0 \Rightarrow f(x)\) 在 \(I\) 上单调递增。
- \(f'(x) < 0 \Rightarrow f(x)\) 在 \(I\) 上单调递减。
- \(f'(x) = 0\) 的点为驻点（可能是极值点）。

## 极值判据

第一充分条件：\(f'(x)\) 在 \(x_0\) 两侧变号，则 \(x_0\) 是极值点。
第二充分条件：\(f'(x_0) = 0\) 且 \(f''(x_0) \neq 0\)，则 \(f''(x_0) > 0\) 时为极小值，\(f''(x_0) < 0\) 时为极大值。

## 凹凸性与拐点

- \(f''(x) > 0\)：曲线凹（向上凹，U 形）。
- \(f''(x) < 0\)：曲线凸（向下凹，∩ 形）。
- 拐点：凹凸性改变的点，\(f''(x) = 0\) 且 \(f''(x)\) 变号。

## 示例：\(f(x) = x e^{-x^2}\)

定义域：\(\mathbb{R}\)。奇偶性：\(f(-x) = -f(x)\)，奇函数，图像关于原点对称。

\[
f'(x) = e^{-x^2}(1 - 2x^2)
\]

令 \(f'(x) = 0\)：\(x = \pm \frac{1}{\sqrt{2}}\)。
- \(x < -\frac{1}{\sqrt{2}}\)：\(f'(x) < 0\)，递减。
- \(-\frac{1}{\sqrt{2}} < x < \frac{1}{\sqrt{2}}\)：\(f'(x) > 0\)，递增。
- \(x > \frac{1}{\sqrt{2}}\)：\(f'(x) < 0\)，递减。

极小值：\(f(-\frac{1}{\sqrt{2}}) = -\frac{1}{\sqrt{2e}}\)。极大值：\(f(\frac{1}{\sqrt{2}}) = \frac{1}{\sqrt{2e}}\)。

## 常见误区

- 只看一阶导数零点就断定极值——必须检查变号。
- 混淆凹凸性的定义（不同教材可能定义相反）。
- 在分析函数图像时遗漏定义域限制。
