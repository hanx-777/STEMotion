# 常微分方程入门

## 基本概念

含有未知函数及其导数的方程称为微分方程。方程中出现的最高阶导数的阶数称为微分方程的阶。

## 一阶可分离变量方程

\[
\frac{dy}{dx} = f(x)g(y)
\]

解法：\(\int \frac{dy}{g(y)} = \int f(x) \, dx\)。

## 一阶线性方程

\[
y' + P(x)y = Q(x)
\]

通解公式：
\[
y = e^{-\int P \, dx}\left(\int Q e^{\int P \, dx} \, dx + C\right)
\]

## 二阶常系数齐次方程

\[
y'' + py' + qy = 0
\]

特征方程：\(r^2 + pr + q = 0\)。

- 两不等实根 \(r_1, r_2\)：\(y = C_1 e^{r_1 x} + C_2 e^{r_2 x}\)
- 两相等实根 \(r\)：\(y = (C_1 + C_2 x)e^{rx}\)
- 共轭复根 \(\alpha \pm \beta i\)：\(y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)\)

## 应用

物理中的振动方程、电路中的 RL/RC/RLC 电路、人口增长模型等都可用微分方程描述。

## 常见误区

- 在分离变量时忘记考虑 \(g(y) = 0\) 的情况。
- 使用通解公式时积分计算错误。
- 在二阶方程中忘记根据特征根类型选择解的形式。
