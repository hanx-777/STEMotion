# 教师课堂演示案例

## 演示一：斜抛运动

**目的：** 展示初速度、角度、初始高度对抛体轨迹的影响。

**参数设置：**
- 固定 \(v_0 = 10\,\text{m/s}\)，改变 \(\theta\)：15°、30°、45°、60°、75°
- 固定 \(\theta = 45°\)，改变 \(v_0\)：5、10、15、20 m/s
- 固定 \(v_0\) 和 \(\theta\)，改变 \(h\)：0、1、2、5 m

**讨论要点：**
- \(h = 0\) 时，45° 射程最远。
- \(h > 0\) 时，最优角度小于 45°。
- 不同角度可达到相同射程（互补角，仅限 \(h = 0\)）。

## 演示二：受力分析

**目的：** 展示斜面上物体的受力分解。

**交互元素：** 滑块改变斜面角度 \(\theta\)，实时显示重力分量 \(mg\sin\theta\) 和 \(mg\cos\theta\)，法向力 \(N\)，摩擦力 \(f\)。

## 演示三：圆周运动

**目的：** 展示向心力与速度、半径的关系。

**参数：** 改变转速（角速度）和半径，实时显示向心加速度和向心力大小。

## 演示四：碰撞过程

**目的：** 展示弹性碰撞与完全非弹性碰撞的区别。

**参数：** 改变两物体质量比和初速度，对比碰撞前后动能变化。

## 演示五：简谐振动

**目的：** 展示弹簧振子的位移、速度、加速度随时间的变化。

**动画要素：** 振子运动 + 正弦曲线同步绘制 + 能量柱状图。

## 参考来源与整理说明

本文为大学物理力学课程的原创整理笔记，为教师提供课堂可视化演示的参数设置和讨论要点。
