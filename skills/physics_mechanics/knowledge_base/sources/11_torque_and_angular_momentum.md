# 力矩与角动量

## 力矩

力矩定义为：
\[
\vec{\tau} = \vec{r} \times \vec{F}
\]

大小：\(\tau = rF\sin\theta\)，其中 \(\theta\) 为 \(\vec{r}\) 与 \(\vec{F}\) 的夹角。单位为 N·m。

力矩是使物体产生角加速度的原因，类比力使物体产生线加速度。

## 转动定律

\[
\tau_{\text{net}} = I\beta
\]

合外力矩等于转动惯量乘以角加速度，这是转动版的牛顿第二定律。

## 角动量

质点的角动量：
\[
\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}
\]

刚体绕固定轴转动的角动量：\(L = I\omega\)。

## 角动量守恒

当合外力矩为零时，角动量守恒：
\[
I_1\omega_1 = I_2\omega_2
\]

应用：花样滑冰旋转（收臂加速）、陀螺稳定性、行星轨道。

## 常见误区

- 混淆力矩和力的单位。
- 在角动量守恒中忘记转动惯量可能变化。
- 将力矩方向搞错（右手定则）。
