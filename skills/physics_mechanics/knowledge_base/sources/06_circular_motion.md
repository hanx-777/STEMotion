# 圆周运动

## 匀速圆周运动

物体以恒定速率 \(v\) 沿半径为 \(r\) 的圆周运动时，角速度 \(\omega = v/r\)，周期 \(T = 2\pi r / v = 2\pi / \omega\)。

向心加速度：
\[
a_c = \frac{v^2}{r} = \omega^2 r
\]

方向始终指向圆心。虽然速度大小不变，但方向不断变化，因此存在加速度。

向心力：
\[
F_c = m\frac{v^2}{r} = m\omega^2 r
\]

向心力不是一种新的力，而是由重力、支持力、摩擦力、张力等提供的合力在指向圆心方向的分量。

## 变速圆周运动

变速圆周运动中，加速度有两个分量：

- 向心加速度 \(a_c = v^2/r\)（改变速度方向）
- 切向加速度 \(a_t = dv/dt\)（改变速度大小）

总加速度大小：\(a = \sqrt{a_c^2 + a_t^2}\)。

## 竖直面圆周运动

在竖直面内做圆周运动时，速度大小不断变化。最高点和最低点的力学条件不同：

最高点：\(T + mg = mv^2_{\text{top}}/r\)（绳的张力 \(T \geq 0\)）
最低点：\(T - mg = mv^2_{\text{bot}}/r\)

## 常见误区

- 认为匀速圆周运动没有加速度——方向变化即有加速度。
- 将向心力当作一种独立的力——它是合力的分量。
- 在竖直面问题中忘记重力的参与。
- 混淆角速度和线速度。
