# 一维运动学

## 位移、速度与加速度

位移是矢量，定义为末位置减初位置：\(\Delta x = x - x_0\)。路程是标量，是运动轨迹的总长度。

平均速度定义为：

\[
\bar{v} = \frac{\Delta x}{\Delta t}
\]

瞬时速度是平均速度在 \(\Delta t \to 0\) 时的极限：\(v = \lim_{\Delta t \to 0} \frac{\Delta x}{\Delta t} = \frac{dx}{dt}\)。

加速度定义为速度的变化率：

\[
a = \frac{dv}{dt}
\]

## 匀变速直线运动公式

当初速度为 \(v_0\)、加速度为 \(a\)（恒定）时，位移和速度满足：

\[
x = x_0 + v_0 t + \frac{1}{2}at^2
\]

\[
v = v_0 + at
\]

\[
v^2 = v_0^2 + 2a(x - x_0)
\]

适用条件：加速度恒定、沿直线运动。变量说明：\(x_0\) 为初始位置（m），\(v_0\) 为初速度（m/s），\(a\) 为加速度（m/s\(^2\)），\(t\) 为时间（s）。

## 自由落体与竖直抛体

自由落体是初速度为零、加速度为 \(g \approx 9.8\,\text{m/s}^2\) 的匀加速直线运动。竖直上抛运动可分段分析：上升阶段速度减小至零，下降阶段速度增大。

## 常见误区

- 混淆速度和速率：速度是矢量，速率是标量。
- 加速度为负不一定表示减速，取决于速度方向。
- 在位移-时间图像中，斜率表示速度，而非位移本身。
