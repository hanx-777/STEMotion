# 大学物理力学典型题库

## 运动学题组

**题1：** 一辆汽车从静止出发，以 \(a = 2\,\text{m/s}^2\) 匀加速行驶 10 秒后，立即以 \(a = -4\,\text{m/s}^2\) 减速直到停止。求总位移。

**题2：** 从高度 \(h = 1.20\,\text{m}\) 的平台以 \(v_0 = 8.0\,\text{m/s}\)、\(\theta = 35°\) 斜抛一小球。求落地时间、水平射程、最大高度。

**题3：** 一物体做匀速圆周运动，半径 \(r = 2\,\text{m}\)，周期 \(T = 4\,\text{s}\)。求线速度、角速度、向心加速度。

## 动力学题组

**题4：** 质量 \(m = 5\,\text{kg}\) 的物体放在倾角 \(30°\) 的粗糙斜面上，\(\mu_k = 0.2\)。求物体沿斜面下滑的加速度。

**题5：** 用轻绳连接两个物体 \(m_1 = 3\,\text{kg}\)、\(m_2 = 5\,\text{kg}\)，放在光滑水平面上，用水平力 \(F = 16\,\text{N}\) 推 \(m_1\)。求绳的张力和系统加速度。

## 能量题组

**题6：** 一个 \(2\,\text{kg}\) 的物体从 \(5\,\text{m}\) 高处自由落下，求落地时的速度（用机械能守恒）。

**题7：** 弹簧（\(k = 200\,\text{N/m}\)）被压缩 \(0.1\,\text{m}\)，弹出一个 \(0.5\,\text{kg}\) 的物体。求物体获得的最大速度。

## 动量题组

**题8：** 一个 \(0.1\,\text{kg}\) 的球以 \(6\,\text{m/s}\) 撞击静止的 \(0.3\,\text{kg}\) 球，弹性碰撞。求碰撞后两球的速度。

**题9：** 一枚 \(0.05\,\text{kg}\) 的子弹以 \(400\,\text{m/s}\) 射入一个 \(2\,\text{kg}\) 的木块（静止），嵌入其中。求共同速度和损失的动能。

## 综合题组

**题10：** 一个 \(1\,\text{kg}\) 的物体从光滑斜面顶端（高 \(3\,\text{m}\)）滑下，在水平面上压缩弹簧（\(k = 500\,\text{N/m}\)）。求弹簧最大压缩量。
