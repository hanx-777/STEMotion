# 矢量与单位

## 基本物理量与单位

国际单位制（SI）中，力学的三个基本量是长度（米，m）、质量（千克，kg）和时间（秒，s）。所有其他力学量均可由这三个基本量导出。例如速度的量纲为 \([v] = \text{m}\cdot\text{s}^{-1}\)，加速度的量纲为 \([a] = \text{m}\cdot\text{s}^{-2}\)。

量纲分析是检验公式正确性的重要手段。一个正确的物理公式，等号两侧的量纲必须一致。例如位移公式 \(x = v_0 t + \frac{1}{2}at^2\) 中，\(v_0 t\) 的量纲为 \(\text{m}\cdot\text{s}^{-1} \times \text{s} = \text{m}\)，\(\frac{1}{2}at^2\) 的量纲为 \(\text{m}\cdot\text{s}^{-2} \times \text{s}^2 = \text{m}\)，两者一致。

## 矢量与标量

标量只有大小，如质量、温度、能量。矢量既有大小又有方向，如位移、速度、加速度、力。矢量用带箭头的符号表示，如 \(\vec{v}\)，其大小记为 \(|\vec{v}|\) 或 \(v\)。

## 矢量分解与合成

在直角坐标系中，矢量 \(\vec{A}\) 可分解为：

\[
\vec{A} = A_x \hat{i} + A_y \hat{j}
\]

其中 \(A_x = A\cos\theta\)，\(A_y = A\sin\theta\)，\(\theta\) 为矢量与 x 轴正方向的夹角。矢量的大小为：

\[
A = \sqrt{A_x^2 + A_y^2}
\]

两个矢量的合成使用平行四边形法则或三角形法则。在坐标系中，对应分量直接相加即可。

## 常见误区

- 将矢量当作标量处理，忽略方向性。
- 角度量纲混淆：弧度（rad）是无量纲量，度（°）需要转换。
- 在斜面问题中，混淆"沿斜面方向"与"水平方向"的分量分解。
