# 摩擦与斜面

## 静摩擦力

静摩擦力 \(f_s\) 的大小在 0 到最大值之间变化：

\[
0 \leq f_s \leq \mu_s N
\]

其中 \(\mu_s\) 为静摩擦系数，\(N\) 为法向力。静摩擦力的方向与相对运动趋势方向相反。最大静摩擦力 \(f_{s,\max} = \mu_s N\) 是物体即将滑动时的临界值。

## 动摩擦力

物体滑动时，动摩擦力为：

\[
f_k = \mu_k N
\]

\(\mu_k\) 为动摩擦系数，通常 \(\mu_k < \mu_s\)。方向与相对运动方向相反。

## 斜面上的受力分析

质量为 \(m\) 的物体放在倾角为 \(\theta\) 的斜面上，重力分解为：

- 沿斜面方向分量：\(mg\sin\theta\)（向下）
- 垂直斜面方向分量：\(mg\cos\theta\)（压向斜面）

法向力：\(N = mg\cos\theta\)（无其他竖直外力时）。

物体沿斜面下滑的加速度（无摩擦时）：
\[
a = g\sin\theta
\]

有摩擦时：
\[
a = g\sin\theta - \mu_k g\cos\theta = g(\sin\theta - \mu_k \cos\theta)
\]

## 临界角

物体刚好开始滑动时，\(\mu_s = \tan\theta_c\)，即 \(\theta_c = \arctan\mu_s\)。

## 单位检查

摩擦系数 \(\mu\) 无量纲。力的单位为 N（kg·m/s\(^2\)）。角度可用度或弧度。

## 常见误区

- 认为静摩擦力总是等于 \(\mu_s N\)——实际只在临界状态时才等于。
- 在斜面上忘记将重力分解，直接用 \(mg\) 作为法向力。
- 混淆静摩擦系数和动摩擦系数。
- 忘记摩擦力方向取决于相对运动趋势。
