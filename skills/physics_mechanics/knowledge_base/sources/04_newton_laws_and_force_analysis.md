# 牛顿定律与受力分析

## 牛顿第一定律

物体在不受外力（或合外力为零）时，保持静止或匀速直线运动状态。这定义了惯性参考系。质量是惯性大小的量度。

## 牛顿第二定律

\[
\vec{F}_{\text{net}} = m\vec{a}
\]

合外力等于质量乘以加速度。适用条件：惯性参考系、质点模型、低速（远小于光速）。变量说明：\(F_{\text{net}}\) 为合外力（N），\(m\) 为质量（kg），\(a\) 为加速度（m/s\(^2\)）。

在直角坐标系中分解为分量形式：
\[
F_x = ma_x, \quad F_y = ma_y
\]

## 牛顿第三定律

两个物体之间的作用力和反作用力大小相等、方向相反、作用在不同物体上。注意：作用力和反作用力不能互相抵消，因为它们作用在不同物体上。

## 受力分析步骤

1. 确定研究对象（隔离体）。
2. 画出重力 \(mg\)，方向竖直向下。
3. 画出接触力：支持力（法向力）垂直于接触面，摩擦力沿接触面。
4. 画出其他外力（绳的张力、弹簧力、电场力等）。
5. 建立坐标系，将各力分解到坐标轴方向。
6. 对每个方向列牛顿第二定律方程。

## 常见力

- 重力：\(G = mg\)，方向竖直向下。
- 支持力（法向力）：垂直于接触面，大小由运动状态决定。
- 弹簧力：\(F = -kx\)（胡克定律），方向指向平衡位置。
- 绳的张力：沿绳方向，忽略绳质量时绳上各处张力相等。

## 常见误区

- 把"匀速运动"等同于"不受力"——匀速运动可以是合外力为零。
- 作用力和反作用力作用在不同物体上，不能合成。
- 在斜面问题中，将重力分量与法向力混淆。
- 忘记考虑加速度方向，导致方程符号错误。
