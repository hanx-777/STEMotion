# 图算法

## 图的表示

邻接矩阵：\(O(V^2)\) 空间，适合稠密图。邻接表：\(O(V+E)\) 空间，适合稀疏图。C++ 中可用 `vector<vector<int>>` 或 `unordered_map<int, vector<int>>`。

## 深度优先搜索（DFS）

从起点出发，沿一条路径尽可能深地探索，回溯后探索其他路径。使用递归或栈实现。

时间复杂度：\(O(V+E)\)。应用：连通分量、拓扑排序、环检测、路径搜索。

## 广度优先搜索（BFS）

从起点出发，先访问所有距离为 1 的邻居，再访问距离为 2 的邻居，依此类推。使用队列实现。

时间复杂度：\(O(V+E)\)。应用：最短路径（无权图）、层序遍历。

## 拓扑排序

对有向无环图（DAG）进行线性排序，使得对每条边 \((u,v)\)，\(u\) 排在 \(v\) 前面。

方法一：Kahn 算法（BFS），不断移除入度为 0 的节点。
方法二：DFS，按完成时间逆序输出。

## 并查集

用于维护不相交集合的合并与查询操作：
- `find(x)`：查找 x 所属集合的代表元素，路径压缩优化后近似 \(O(1)\)。
- `union(x, y)`：合并 x 和 y 所在集合，按秩合并优化后近似 \(O(1)\)。

## 常见误区

- 在 DFS 中忘记标记已访问节点导致无限循环。
- 在 BFS 中忘记使用队列而错误使用栈。
- 混淆有向图和无向图的邻接表表示。
