# 栈与队列

## 栈（Stack）

栈是后进先出（LIFO）的线性数据结构。核心操作：
- `push(x)`：将元素 x 压入栈顶，时间复杂度 \(O(1)\)。
- `pop()`：弹出栈顶元素，时间复杂度 \(O(1)\)。
- `top()` 或 `peek()`：查看栈顶元素但不弹出，\(O(1)\)。
- `empty()`：判断栈是否为空，\(O(1)\)。

C++ 实现：`std::stack<T>`，底层默认使用 `std::deque`。

## 队列（Queue）

队列是先进先出（FIFO）的线性数据结构。核心操作：
- `push(x)`：将元素 x 加入队尾，\(O(1)\)。
- `pop()`：弹出队头元素，\(O(1)\)。
- `front()`：查看队头元素，\(O(1)\)。
- `empty()`：判断队列是否为空，\(O(1)\)。

C++ 实现：`std::queue<T>`，底层默认使用 `std::deque`。

## 单调栈

单调栈是一种维护栈内元素单调性的技巧。典型应用：求每个元素右边第一个比它大的元素。

算法流程：
1. 从右向左遍历数组。
2. 维护一个单调递减栈。
3. 对于当前元素，弹出栈中所有小于等于它的元素。
4. 栈顶即为右边第一个比它大的元素。
5. 将当前元素压入栈。

时间复杂度：\(O(n)\)，因为每个元素最多入栈和出栈各一次。

## 常见误区

- 在栈为空时调用 `pop()` 或 `top()` 导致未定义行为。
- 混淆栈和队列的特性。
- 在单调栈实现中忘记处理栈为空的边界情况。
